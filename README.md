## Usage

Simply login to MySQL in the terminal
```
mysql -u root -p
```

Source the database
```
source db/schema.sql
```

Run the seed files
```
npm run seed
```

Then start the application
```
npm run start
```

From there I tested all the API endpoints with insomnia
